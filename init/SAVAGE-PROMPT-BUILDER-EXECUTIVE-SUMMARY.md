# Savage Prompt Builder — Executive Summary

## What It Is

Savage Prompt Builder is an end-to-end AI image creation platform. Users build prompts, generate images, and manage their creative output all in one app. The app handles the hard part — prompt engineering — so the user focuses on creative decisions and gets the best possible results from AI image generation.

The difference between Savage and other AI image tools: we don't just give users a text box and a Generate button. We guide them through a structured creative process using specialized templates, 300+ curated style presets, 108 color palettes, and an AI agent that optimizes everything before the image is ever generated. The prompt is perfect before the user hits Generate, which means fewer wasted generations, better results on the first try, and a faster creative workflow.

Think of it like Higgsfield for AI images — except every generation starts with the best possible prompt, built by an intelligent system that knows exactly how to talk to each AI model.

## Who It's For

**Primary users:** Designers, brand owners, print-on-demand sellers, and creative entrepreneurs who create visual content at scale using AI image generation.

**Specific use cases:**
- A streetwear brand owner designing t-shirt graphics — builds the prompt, generates the image, previews it on a mockup of a black hoodie, saves it to a project folder for their summer collection
- A social media manager creating content across platforms — picks "Instagram Reel" from the platform dropdown, the app auto-sets the right dimensions, generates the image at the correct aspect ratio
- A tattoo artist generating concept art for clients — builds the design, previews it on a forearm placement mockup, iterates through variations, sends the final to the client
- An indie musician creating album cover artwork — picks the Album Cover template, selects a mood and genre aesthetic, generates 4 options, saves the winner to their media gallery
- A sticker shop owner producing die-cut designs — builds the prompt optimized for flat colors and thick outlines, generates the image, knows it's production-ready because the template enforced the right constraints

These users generate dozens or hundreds of AI images per week. They need the best possible results with the least friction, and they need everything in one place — not scattered across Midjourney's Discord, DALL·E's web app, and a folder of text files.

## The Problem It Solves

AI image generation is powerful but the workflow is broken. Users bounce between tools — they write prompts in a notes app, paste them into a generator, save results to a downloads folder, lose track of what settings produced their best work, and start from scratch every time. The prompts themselves are the biggest bottleneck: each generator has different syntax, different strengths, different character limits, and different ways of handling style direction. Most users never learn the prompt engineering techniques that separate mediocre results from exceptional ones.

Savage Prompt Builder collapses the entire workflow into one app. Build the prompt with intelligent guidance, generate the image with the right model, organize everything in one library. The prompt engineering knowledge is baked into the system — users get professional-quality prompts without having to become prompt engineers themselves.

## How It Works

1. **Pick a template.** 20 templates, each designed for a specific creative use case — Clothing, Social Media, Brand/Logo, Tattoo, Album Cover, Sneaker, Pattern, Character, and more. Each template knows what fields matter, what styles work, and how to structure the prompt for that type of image.

2. **Fill in the creative brief.** Each template presents 5-7 fields like Subject, Style, Mood, Colors, and Composition. Suggestion chips provide instant options. The Social Media template includes a dropdown with 40+ platform-specific aspect ratios that auto-inject the correct dimensions.

3. **Layer on styles, palettes, and keywords.** Activate visual style presets (over 300 curated for each template type), select from 108 color palettes, and add keyword tags. Everything weaves into the prompt automatically and updates the preview in real time.

4. **Let the AI agent optimize.** The in-app AI prompting agent — trained in best practices for every supported generator — polishes the prompt, suggests field values based on creative direction, generates custom style presets from a vibe description or mood board, recommends the best generator for the desired result, auto-builds negative prompts, checks for style conflicts, and creates variations that explore different creative angles. All of this happens before a single image is generated, so the prompt is as good as it can be when the user hits Generate.

5. **Generate.** Hit Generate and the image is created directly in the app. Choose from 8 models — NanoBanana Pro (default), Midjourney, DALL·E 3, Stable Diffusion XL, Flux Pro, Leonardo AI, Adobe Firefly, or Ideogram. Select image count (1-4) and aspect ratio. Results appear in the app with save, download, and copy actions.

6. **Iterate and refine.** Not quite right? Iterate on the prompt (the app tracks version chains: v1 → v2 → v3), remix the style setup, or ask the AI agent for variations. Generate again. The feedback loop stays inside the app.

7. **Organize everything.** Generated images go to the Media Gallery with full metadata. Prompts save to History with star ratings, notes, and project folders. Entire builder configurations save as Recipes that restore every setting with one click. Bulk mode generates dozens of prompts from a subject list for batch production.

**AI Creative Agent (Pro tier)**

The app includes a built-in AI prompting agent trained in best practices for every supported image generator. This is the core differentiator — the agent optimizes prompts before images are generated, so users get better results on the first try.

- AI Polish — the agent rewrites prompts using best practices specific to both the template type and the target generator. It restructures for the generator's strengths, tightens redundant phrasing, adds precise visual language, and enhances lighting/composition direction.
- AI Variations — generates 3-5 alternative creative directions from a single prompt, each genuinely different (not synonym swaps)
- AI Suggest — analyzes filled fields and suggests values for empty ones that create a cohesive creative direction. If the user entered "skull" with "screenprint" style, the agent suggests complementary mood and technique values.
- AI Style Generator — describe a vibe or upload mood board images, the agent creates 3 custom style presets with detailed technique terms, color references, and artistic references tailored to the template
- Auto Generator Selection — the agent analyzes the prompt and recommends the best model for the job (photo-realistic → Flux, stylized illustration → Midjourney, text-heavy → Ideogram)
- Negative Prompt Optimization — the agent auto-builds targeted negative prompts considering the specific subject, style, and generator
- Style Combination Advisor — warns when active styles conflict and suggests pairings that work well together
- Iterative Refinement — user describes what's wrong with a generated image in plain language, the agent rewrites the prompt to fix it
- Collection Coherence — for series and collections, the agent ensures visual consistency across multiple prompts

## Revenue Model

Freemium tier system:

**Free tier** — full prompt building, styling, saving, and organization. Users can build prompts, copy them, save to history, manage projects, and use all 20 templates, 300+ styles, 108 palettes, and 140+ keywords without paying. This gets users in the door and building a library they don't want to leave.

**Pro tier** — unlocks the AI agent (Polish, Variations, Suggest, Style Generator) and in-app image generation. These are the features that make the workflow truly end-to-end. When a free user clicks any Pro feature, an upgrade modal shows what they're missing.

The conversion path: users start on Free, build up a library of saved prompts and recipes, realize they're still copy-pasting to external generators, and upgrade to Pro to keep everything in one place.

## Technical Overview

**Current state:** A fully functional 4,600-line HTML/JS prototype with all features working. A React/JSX rebuild exists with the UI direction but needs completion.

**Architecture:** Single-page application with 10 views. State managed in memory with localStorage persistence. AI features route through a backend API proxy to OpenAI (GPT-4o-mini for text) and NanoBanana Pro (for image generation). The app auto-detects backend availability.

**Data:** All template definitions, style presets, field suggestions, palettes, keywords, negative presets, mockup configurations, generator formats, AI system prompts, and aspect ratios are stored in JSON data files.

**Target production stack:** React/Next.js frontend, Node.js backend with API proxy, database for user accounts and cloud sync.
