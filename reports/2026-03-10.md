### Session 8 — Bug Fixes

Found and fixed 3 production-blocking bugs during testing:

| Bug | Root Cause | Fix |
|-----|-----------|-----|
| Settings page infinite re-render crash | Zustand selectors creating new object refs every render | Wrapped with `useShallow()` from `zustand/react/shallow` |
| Production build fails: `useSearchParams()` not in Suspense | Next.js 16 requires Suspense boundary for static generation | Added `<Suspense>` wrapper, extracted `BuilderPageInner` |
| Production build fails: API route params type error | Next.js 16 params are async `Promise<>` | Updated type + added `await params` |

Production build verified clean — all 18 routes compiled, zero errors.
