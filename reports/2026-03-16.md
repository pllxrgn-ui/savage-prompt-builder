## Technical Notes

### Pollinations.ai API
- **Working endpoint**: `https://gen.pollinations.ai/image/{encoded_prompt}?width=W&height=H&model=MODEL&nologo=true&seed=S`
- **Auth**: `Authorization: Bearer {key}` header
- **Best model**: `gptimage` (DALL-E based, excellent prompt following, 80-420 KB images)
- **Avoid**: `image.pollinations.ai/prompt/` (returns 500s), `flux` model (low quality 6-44 KB)
- **Valid models**: kontext, nanobanana, flux, gptimage, imagen-4, seedream5, dirtberry-pro

### Next.js Image Caching
- Dev server keeps an **in-memory cache** of `/_next/image` optimized images
- Replacing files on disk is NOT enough — requires full `.next` directory deletion + dev server restart
- Clearing only `.next/cache/images` is insufficient
